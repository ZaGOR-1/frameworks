# Bank CRUD API

У проєкті додано CRUD API для теми Bank у Laravel та Symfony.

## Endpoint-и

- GET /api/v1/bank-accounts
- GET /api/v1/bank-accounts/{id}
- POST /api/v1/bank-accounts
- PATCH /api/v1/bank-accounts/{id}
- DELETE /api/v1/bank-accounts/{id}

## Postman collections

- laravel/postman/laravel-bank-api.postman_collection.json
- symfony/postman/symfony-bank-api.postman_collection.json
